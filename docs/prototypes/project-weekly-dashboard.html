项目周报

1. 项目周报总览

围绕进度、风险、质量、测试、资源和行动闭环给出本周管理驾驶舱。

当前判断：重点关注

2. 项目健康量化表

六个维度统一量化，便于管理层快速判断项目状态和优先动作。

维度 核心指标 趋势 判断 建议动作

3. 本周进展与交付

展示 PM 周进展、交付范围、近期演示和关键里程碑。

交付范围

近期重点演示

4. 里程碑状态

承诺交付、实际完成和延期风险。

里程碑 承诺时间 状态 负责人

5. 风险与行动闭环

聚焦本周需要推动闭环的风险项、责任人和截止时间。

风险项 影响范围 责任人 截止时间 当前状态 管理层支持

6. 质量与测试

缺陷、严重问题、测试执行和阻塞用例共同判断质量状态。

模块 用例数 已执行 通过率 失败/阻塞 质量结论

7. 负责人 / 团队排行

用于发现闭环压力、缺陷归属、延期和资源瓶颈。

8. 近 7 日趋势

来自每日快照，展示本周趋势和环比变化。
